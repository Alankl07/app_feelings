# App Sentimentos

### Passo á passo para executar o app em sua máquina


```bash
# Clonando o projeto.
$ git clone https://github.com/Alankl07/app_feelings.git

# Acessando a pasta do app.
$ cd app_feelings

# Instalando as dependências.
$ yarn install

# Executando a aplicação.
$ expo start 
```
